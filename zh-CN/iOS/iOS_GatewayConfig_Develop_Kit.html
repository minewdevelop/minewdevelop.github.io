        # GatewayConfig SDK 说明文档

        # 前言

        GatewayConfig APP 又名网关助手，由深圳云里物里科技股份有限公司开发完成并发布，其主要功能是用来进行对深圳云里物里科技股份有限公司所生产的相关网关设备进行参数配置。

        # WiFi 配置功能

        WiFi 配置功能使用了乐鑫科技股份有限公司的 SDK ，其功能主要是用于手机 APP 进行 WiFi 广播数据，以及接收 WiFi 广播数据使用，其 SDK 地址为：

        ESP-WiFi 广播 SDK 地址 (opens new window)

        在 Github 地址里边的 README.md 中，查看 EspTouchV2 的文档；

        # API 接口

        API 接口是网关设备作为 HTTP 服务端，手机 APP 在通过 WiFi 配网之后获得网关设备地址，并且手机与网关处于同一局域网下的前提下，通过 API 接口来进行对网关设备的参数修改；
